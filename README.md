## ✨ 产品简介

**CodeBaseAsk** 是一款桌面端 AI 智能代码助手，帮助开发者快速理解和分析任意代码仓库。无论是开源项目还是本地项目，只需简单操作即可生成详细的项目文档 Wiki，并通过自然语言与 AI 进行深度问答交互。

---

## 🚀 核心功能

### 📚 智能 Wiki 文档生成

自动分析代码结构，生成结构化的项目文档，帮助快速了解项目全貌。

- **快速模式**：5-10 分钟生成简要概览文档
- **详细模式**：30 分钟生成完整技术文档
- **自动目录**：智能提取章节结构，支持跳转导航
- **图表渲染**：支持 Mermaid 流程图、时序图等可视化展示
- **收藏功能**：一键收藏常用项目，快速访问

![image-20260111222903265](http://devyk.top/2022/202601112229112.png)

---

### 💬 AI 智能问答

基于代码库上下文，与 AI 进行深度对话，快速获取代码相关问题的答案。

- **上下文感知**：AI 理解整个代码库结构和逻辑
- **实时流式响应**：打字机效果，即时查看回答
- **工具调用展示**：透明展示 AI 调用的分析工具
- **Markdown 渲染**：代码高亮、表格、图表完美呈现
- **历史记录**：保留对话历史，随时回顾

### ⚙️ LLM 模型自定义配置

完全自定义 AI 模型设置，支持接入各类大语言模型服务。

- **模型支持**：兼容 OpenAI 模型，后续会适配更多模型
- **自定义 API**：配置任意兼容 OpenAI 协议的 API 端点

- **本地配置**：所有配置信息安全存储在本地

## 🏆 产品优势

### 与其他代码分析产品对比

| 特性             | CodeBaseAsk        | 在线代码分析工具 | IDE 插件     |
| ---------------- | ------------------ | ---------------- | ------------ |
| **数据隐私**     | ✅ 代码数据本地管理 | ❌ 代码上传云端   | ⚠️ 部分上传   |
| **实时问答**     | ✅ 基于代码实时回答 | ⚠️ 静态分析       | ⚠️ 有限上下文 |
| **完整项目分析** | ✅ 全仓库深度理解   | ⚠️ 文件级分析     | ⚠️ 项目级受限 |
| **自定义模型**   | ✅ 灵活配置         | ❌ 固定模型       | ⚠️ 有限选择   |

### 🔒 数据安全优势

CodeBaseAsk 采用**本地优先**架构：

- **代码不上传**：所有代码分析在本地完成，代码内容不会传输到第三方服务器
- **隐私可控**：仅将必要的问答请求发送给您配置的 LLM 服务
- **适合企业**：满足企业级代码安全合规要求

### ⚡ 实时问答优势

不同于静态文档，CodeBaseAsk 提供真正的**代码感知问答**：

- **代码上下文理解**：AI 能够读取和分析代码文件内容
- **跨文件关联**：理解函数调用、模块依赖等代码关系
- **实时更新**：代码修改后问答结果实时反映最新状态
- **精准定位**：可以指向具体的代码位置和实现细节

## 📦 安装

### macOS

下载链接: https://github.com/yangkun19921001/CodeBaseAsk/releases/tag/v1.0.0

1. 下载 DMG 安装包：
   - Intel 芯片：`CodeBaseAsk-1.0.0.dmg`
   - Apple Silicon：`CodeBaseAsk-1.0.0-arm64.dmg`
2. 打开 DMG 文件，将应用拖入 Applications 文件夹
3. 首次运行时右键选择"打开"

### Windows(后续支持)

1. 下载 `CodeBaseAsk-Setup-1.0.0.exe`
2. 双击运行安装程序
3. 按提示完成安装

### Linux(后续支持)

1. 下载 `CodeBaseAsk-1.0.0.AppImage`
2. 添加执行权限：`chmod +x CodeBaseAsk-1.0.0.AppImage`
3. 双击或命令行运行

## 🛠️ 快速开始

1. **启动应用**：首次启动会自动初始化后端服务
2. **配置 LLM**: 生成 wiki 之前必须要配置 llm 大语言模型，目前仅仅支持 openai 格式的 api
3. **添加仓库**：
   - 输入 GitHub/Gitee 仓库 URL，或
   - 点击"添加本地项目"选择文件夹
4. **生成 Wiki**：选择快速/详细模式，等待生成完成
5. **开始问答**：点击右上角"Ask AI"按钮，输入问题
